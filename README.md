# PetShop ¡La tienda para tus mascotas!

Página web de PetShop utilizando el framework Django utilizando varias librerías

# Dependencias

Django:
```
pip install django
```

Crispy Forms:
```
pip install django-crispy-forms
```

Boostrap4:
```
pip install bootstrap4
```

Pillow:
```
pip install pillow
```
