# PetShop ¡La tienda para tus mascotas!

Página web de PetShop realizada con el framework Django y con la utilización de varias librerías

# Dependencias

Django:
```
pip install django
```

django-crispy-forms:
```
pip install django-crispy-forms
```

Boostrap4:
```
pip install bootstrap4
```

Pillow:
```
pip install pillow
```
